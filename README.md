# IPL-Data-Analysis-Dashboard
This project is an IPL Data Analysis Dashboard created using Power BI.It analyzes IPL match and player data to show team performance, player statistics, and match outcomes through interactive charts and filters.

Project Objective
       Analyze IPL match data
       Compare team and player performance
       Show insights using charts and visuals
       Learn Power BI concepts like data cleaning and visualization

Tools Used
     Power BI Desktop
     Power Query
     DAX
     Excel / CSV

Data Cleaning
     Removed empty and duplicate values
     Fixed column names
     Cleaned data using Power Query

Dashboard Features
     Filters for season, team, and player
     Charts for team performance
     Player runs and wickets analysis
     Match and venue analysis

Insights
    Best performing teams
    Top batsmen and bowlers
    Match results based on venues
    Impact of toss on match outcome
